RestfulWordnet-API
==================

Custom Wordnet offering as RESTful API

* **Wordnet Offering**
  - synonyms
  - hypernyms
  - hyponyms

<blockquote>
curl -X GET -H "Accept: application/json" -H "Content-type: text/plain" http://localhost:10000/RestfulWordnet/api/wordnet/all/education
</blockquote>

* **Supported Formats**
  - application/json
  - application/xml
